# Pim tilemaps

A tilemap editor for creating pixel worlds!

## Features

- Fixed size grid max 250 * 250 tiles
- Multiple layers
- Paint and erase tile(s)
- Rule based 8-direction auto tiles
- Mark tiles with user defined areas
- Manage tile attributes
- Download and Upload project
- Export to JSON

## Tech stack 
- [Svelte](https://svelte.dev/)
- [Canvas 2D](https://developer.mozilla.org/en-US/docs/Web/API/Canvas_API)
